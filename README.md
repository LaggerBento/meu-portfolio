# 🌐 My Project Hub - LaggerBento

Este repositório serve como a porta de entrada principal para os meus projetos desenvolvidos durante o curso de Desenvolvimento de Sistemas. É um portal centralizado que organiza e facilita o acesso às minhas aplicações web.

## 📍 Projetos em Destaque

### 🎵 Player de Música Gospel
* **Foco:** Infraestrutura e Performance.
* **Destaque:** Armazenamento em nuvem via Cloudinary e processamento de áudio via Web Audio API.
* **Visual:** Design Neon com visualizador de frequências dinâmico.

### 🎮 Game Music Player
* **Foco:** Lógica de Programação e Temas.
* **Destaque:** Troca dinâmica de playlists e estilos (Half-Life e Minecraft) usando estados em JavaScript.
* **Visual:** Interface inspirada em interfaces clássicas de jogos.

## 🛠️ Tecnologias Utilizadas neste Portal
- **HTML5 Profissional:** Semântica e SEO.
- **CSS3 Avançado:** Efeitos de Glassmorphism (vidro) e total responsividade para dispositivos móveis.
- **Git & GitHub:** Gestão de múltiplos repositórios e deploy via GitHub Pages.
